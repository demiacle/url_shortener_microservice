Create a (hopefully) shorter url!

*if a short url has already been created, you will receive the already created link

*urls must be either http:// or https://

Example creation usage:

https://neat-snail.glitch.me/new/http://foo.com:80
https://neat-snail.glitch.me/new/https://www.google.com

Example creation output

{
"original_url":"https://www.google.com/",
"short_url":"https://neat-snail.glitch.me/crazyword"
}

Usage:

https://neat-snail.glitch.me/crazyword

Will redirect to:

https://www.google.com/
